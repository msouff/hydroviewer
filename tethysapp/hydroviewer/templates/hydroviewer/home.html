This web application was created to allow users to visualize hydrologic model results from different sources. It currently can display ECMWF and LIS model results.

How it works

1. Select a hydrologic model from the Select Model dropdown.
2. Click submit to load the sub page for that specific model.
3. Select a watershed from the Select Watershed dropdown.
4. Once a watershed is selected, the map will automatically zoom to that specific watershed. Click on any of the stream reaches within the watershed.
5. A hydrograph for the selected reach with the most current streamflow result will be displayed in the grapth area next to the map.
6. If previous dates are available, the Select Date dropdown will be displayed on top of the graph. Select any of the available dates to display the forecast for that specific date in the graph.
7. Click this button to return to the main page.
8. Click this button to exit the application.
